## Классификации
```mermaid

	P(Porifera) --> S(Symplasma)
	P --> C(Cellularia)
		S --> Hexactinellida
		C --> Calcispongia
		C --> Demospongia
		C --> Homoscienomorpha
```
У Symplasma почти вся хоанодерма превращается в синцитий
Hexactinellida 
Calcispongia - скелет из $CaCO_3$
Demaspongia - SiO2 + спонгии
